## What This Guide Covers

This guide is written for complete beginners with no prior Photoshop experience. It walks through two core topics in a step-by-step format:

**Non-Destructive Editing**
How to edit images in Photoshop without permanently altering the original pixels, using Adjustment Layers, Smart Objects, and Smart Filters.

**Vector Shapes**
How to create and edit scalable vector graphics in Photoshop using the Shape Tool, Pen Tool, and Direct Selection Tool.

The guide also includes:

- A **Glossary** defining key Photoshop terms used throughout
- A **Troubleshooting Guide** covering the six most common issues beginners encounter, with step-by-step fixes and confirmation notes
- A **Home page** orienting readers before they begin

## Process and Methods

**Writing for the web**
All content follows web writing principles covered in COMM 2116 — plain language, short sentences, chunked steps, and active voice throughout. Each instruction page is structured so a beginner can follow along in Photoshop without needing to refer to any other resource.

**Instructional design**
Each tutorial section follows a consistent structure: an overview, a learning outcomes list, prerequisites, and numbered steps. MkDocs Material admonitions (`!!! note` and `!!! warning`) are used to surface important information without interrupting the flow of the steps.

**Collaborative authoring**
The guide was written in pairs, with each team member responsible for separate sections. Version control was managed through Git using conventional commit messages throughout the project.

**Peer testing**
The guide was reviewed by peers before final submission to verify that a complete beginner could follow each set of instructions without getting stuck. Feedback from peer testing was used to refine unclear steps and add troubleshooting entries.
