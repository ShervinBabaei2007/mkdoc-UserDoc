# Vector Shapes in Photoshop

## Overview

Vectors are shapes defined by mathematical paths rather than pixels, meaning they
can be scaled to any size without losing quality. In Photoshop, you can create
vectors using the **Pen Tool** and **Shape Tool**,  making them ideal for logos,
icons, and precise graphic elements.

## What You Will Learn

By the end of this section, you will be able to:

1. Understand the difference between vector and raster graphics
2. Create a simple vector shape using the Shape Tool
3. Edit and customize vector shapes using the Pen Tool

## Before You Begin

!!! note
    Make sure you have a new blank canvas open in Photoshop before starting.

!!! note
    Vector shapes in Photoshop are stored as **Shape Layers** — they appear
    in your Layers panel just like regular layers.
