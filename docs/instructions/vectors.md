# Vector Shapes in Photoshop

## Overview

Vectors are shapes defined by mathematical paths rather than pixels, meaning they can be scaled to any size without losing quality. Unlike regular photos or raster images, vector shapes will never become blurry or pixelated no matter how large or small you make them. In Photoshop, you can create vectors using the **Shape Tool** and **Pen Tool** — making them ideal for logos, icons, buttons, and any graphic element that needs to look sharp at any size.

## What You Will Learn

By the end of this section, you will be able to:

1. Understand the difference between vector and raster graphics
2. Create a simple vector shape using the Shape Tool
3. Draw a custom vector path using the Pen Tool
4. Adjust and fine-tune your vector shape using the Direct Selection Tool

## Before You Begin

!!! note
    This tutorial will guide you through creating a new canvas from scratch. No existing file is required — just open Photoshop and follow along from Step 1.

!!! note
    Vector shapes in Photoshop are stored as **Shape Layers** — they appear in your Layers panel just like regular layers. This means you can always go back, select the layer, and edit the shape at any time without losing quality.

## Getting Started

This section will walk you through creating and editing vector shapes in Photoshop from scratch. The tutorial is split into two parts — Part 1 covers creating a basic shape using the Shape Tool, and Part 2 covers drawing a fully custom shape using the Pen Tool. Follow each step carefully, as the Pen Tool in particular takes practice to get used to.

---

## Part 1: Creating a Shape Using the Shape Tool

The Shape Tool is the easiest way to create vector shapes in Photoshop. It gives you a selection of pre-built shapes like rectangles, circles, stars, and polygons that you can place onto your canvas instantly without having to draw anything by hand.

### Step 1: Open Photoshop and Create a New File

When you open Photoshop, you will land on the Home screen. From here, you need to create a new blank canvas to work on — think of it as setting up a blank sheet of paper before you start drawing.

1. **Open** Photoshop — you will see the Home screen with options to create or open a file

    ![Photoshop home screen showing New File and Open buttons](../assets/images/shape%20tool/Screenshot%202026-03-10%20191024.png)

    *Figure 1: Photoshop Home screen*

2. **Click** [File] → [New] or press `Ctrl + N` to open the new file dialog

    ![File dropdown menu open with New highlighted at the top](../assets/images/shape%20tool/Screenshot%202026-03-10%20191211.png)

    *Figure 2: Navigating to File → New*

3. **Set** Width to `1920`, Height to `1080`, Resolution to `72`, and Color Mode to `RGB Color`, then **click** [Create]

!!! note
    You can use any canvas size for this tutorial — the default settings work fine for beginners. The canvas size just determines how large your working area is and does not affect the quality of your vector shapes.

---

### Step 2: Select the Shape Tool

The Shape Tool is hidden in the left toolbar alongside several other shape options. You need to right-click the toolbar icon to reveal the full list of available shapes, because each shape type creates a different kind of vector — a rectangle, a circle, a star, and so on.

1. **Right-click** the Shape Tool icon in the left toolbar — it looks like a small rectangle
2. **Select** [Rectangle Tool] from the dropdown menu — you will see options including Ellipse, Triangle, Polygon, Star, Line, and Custom Shape

    ![Shape tool dropdown menu showing Rectangle, Ellipse, Triangle, Polygon, Star, Line and Custom Shape tools](../assets/images/shape%20tool/Screenshot%202026-03-10%20191341.png)

    *Figure 3: Shape Tool options in the toolbar*

!!! note
    You can press `U` on your keyboard to quickly select the Shape Tool. If you press `Shift + U` it will cycle through the different shape options, so you can switch between rectangle, ellipse, star, and more without touching the mouse.

---

### Step 3: Set the Fill Color

Before you draw your shape, it is important to choose a fill color first. The fill color is what appears inside your shape — without it, your shape will look transparent or completely invisible on the canvas.

1. **Click** the **Fill** color box in the top options bar — it is located at the top left of your screen once the Shape Tool is selected
2. The **Color Picker** dialog will open — by default it shows black (`#000000`)

    ![Color Picker dialog open showing default black color with hex field showing 000000](../assets/images/shape%20tool/Screenshot%202026-03-10%20191451.png)

    *Figure 4: Color Picker dialog with default black color*

3. **Type** a hex color code in the `#` field at the bottom (e.g., `3760be` for blue) or **drag** the color sliders to visually pick a color you like
4. **Click** [OK] to confirm your selected color

    ![Color Picker with blue color selected showing hex value 3760be](../assets/images/shape%20tool/Screenshot%202026-03-10%20191639.png)

    *Figure 5: Selecting a blue fill color (#3760be)*

!!! note
    If you do not see the Fill option at the top, make sure the Rectangle Tool is still selected — the options bar changes depending on which tool is active.

!!! note
    The **new** and **current** boxes at the top of the Color Picker let you compare your new color against the previous one before confirming. The small triangle (⚠️) that may appear is Photoshop's out-of-gamut warning — it means the color ma not print accurately, but is safe to ignore for digital projects. Click [OK] to continue.

---

### Step 4: Draw the Shape

Now that your tool and color are set up, you are ready to draw your first vector shape. Drawing a shape in Photoshop is as simple as clicking and dragging across the canvas — Photoshop automatically creates it as a vector Shape Layer that you can resize and edit at any time.

1. **Click and drag** on the canvas to draw your rectangle — the further you drag, the larger the shape will be
2. **Release** the mouse when the shape is the size you want — you will see small blue anchor points appear at the corners and edges, confirming it has been created as a vector

    ![Yellow rectangle drawn on canvas with blue anchor points visible at corners and edges](../assets/images/shape%20tool/Screenshot%202026-03-10%20191948.png)

    *Figure 6: A rectangle vector shape drawn on the canvas*

!!! note
    Hold `Shift` while dragging to constrain the shape proportions — this forces a rectangle to become a perfect square, or an ellipse to become a perfect circle. Release `Shift` after you release the mouse to keep the proportions locked.

!!! warning
    Make sure the mode in the top options bar is set to **Shape** and not **Path** or **Pixels** — otherwise your shape will not be saved as a vector layer and you will lose the ability to scale it without quality loss.

## Part 2: Drawing a Custom Path Using the Pen Tool

The Pen Tool is one of the most powerful — and most challenging — tools in Photoshop. Unlike the Shape Tool which gives you pre-made shapes, the Pen Tool lets you draw any shape you can imagine by manually placing individual anchor points on the canvas. Each anchor point is connected to the next by a path, and together they form the outline of your fully custom vector shape.

### Step 5: Select the Pen Tool

The Pen Tool is located in the left toolbar and has several sub-tools hidden underneath it. For this tutorial, you will use the standard **Pen Tool**, which gives you the most control over your paths and anchor points compared to the other Pen Tool variations.

1. **Right-click** the Pen Tool icon in the left toolbar — it looks like the tip of a fountain pen nib
2. **Select** [Pen Tool] from the dropdown menu — you will see options for Freeform Pen, Curvature Pen, and Anchor Point tools

    ![Pen Tool dropdown showing Pen Tool, Freeform Pen Tool, Curvature Pen Tool, Add Anchor Point Tool, Delete Anchor Point Tool and Convert Point Tool](../assets/images/Pen%20tool/Screenshot%202026-03-10%20192138.png)

    *Figure 7: Pen Tool options in the toolbar*

!!! note
    You can press `P` on your keyboard to quickly select the Pen Tool. The other tools in the dropdown (like Curvature Pen and Freeform Pen) are easier to use but give you less precise control — stick to the standard Pen Tool for now until you are comfortable with the basics.

---

### Step 6: Draw Anchor Points to Create a Path

The Pen Tool works very differently from a regular drawing tool. Instead of drawing freely, you place individual **anchor points** on the canvas one click at a time, and Photoshop draws a straight or curved path between each one. Understanding the difference between a regular click and a click-and-drag is the most important concept to grasp when using the Pen Tool.

!!! warning
    Before clicking, make sure the mode in the top options bar is set to **Shape** and not **Path** — otherwise your pen path will not create a filled vector layer.

  ![Top options bar showing Shape mode selected](../assets/images/Pen%20tool/image.png)

  *Figure 8: Top options bar set to Shape mode*

1. **Click** anywhere on the canvas to place your first anchor point — this is the starting point of your shape
2. **Click** a second location on the canvas to place another anchor point — Photoshop will automatically draw a straight line connecting the two points
3. **Click and drag** instead of just clicking to create a **curved** path between points — as you drag, you will see a curve handle appear that controls the direction and intensity of the curve

    ![Two anchor points on the canvas connected by a smooth curved path with direction handles visible](../assets/images/Pen%20tool/Screenshot%202026-03-10%20192407.png)

    *Figure 9: A curved path created with two anchor points*

!!! note
    **Clicking** creates a sharp corner point, while **click and dragging** creates a smooth curved point. Most custom shapes use a combination of both — sharp corners for geometric areas and smooth curves for organic, flowing sections. It may take a few tries to get a feel for how the curve handles work, so do not get discouraged if it feels awkward at first.

---

### Step 7: Continue Adding Points to Build Your Shape

Once you have placed your first few anchor points, you can continue building your shape by adding more. The more anchor points you place, the more detailed and complex your shape can become — think of it like connecting the dots, where each click adds a new dot and the path follows behind connecting them all.

1. **Continue clicking** around the canvas to add more anchor points in the direction you want your shape to go
2. **Click and drag** at each new point to add smooth curves wherever your shape needs to turn or flow naturally
3. **Keep adding** points until your shape outline is almost complete — you should be able to see the rough shape forming from the connected paths

    ![Multiple anchor points on canvas forming an incomplete curved blob shape](../assets/images/Pen%20tool/Screenshot%202026-03-10%20192506.png)

    *Figure 10: Building a custom path with multiple anchor points*

!!! note
    Do not worry if your first attempt does not look perfect — the Pen Tool genuinely takes practice and even experienced designers find it tricky at first. You can always go back and adjust individual anchor points after you close the path, which is covered in Step 9.

---

### Step 8: Close the Path

An open path is just a line — it has a starting point and an ending point that are not connected to each other. To turn your path into a proper closed shape that can be filled with a solid color, you need to connect your last anchor point back to the very first one you placed. This action is called **closing the path** and is required before you can apply any fill color.

1. **Hover** your cursor slowly over the very first anchor point you placed — watch carefully for a small circle icon to appear next to the pen cursor, which signals that you are positioned correctly to close the path
2. **Click** on the first anchor point to close the path and complete your shape

    ![Nearly closed path with cursor hovering near the starting anchor point](../assets/images/Pen%20tool/Screenshot%202026-03-10%20192634.png)

    *Figure 11: Hovering over the first anchor point to close the path*

    ![Fully closed path](../assets/images/Pen%20tool/Screenshot%202026-03-10%20192716.png)

    *Figure 12: A completed closed vector path*

3. **Confirm** the path is now fully closed — it will form a complete enclosed shape with no open gaps

!!! warning
    If you do not close the path, Photoshop will treat it as an open line rather than a filled shape. Always look for the small circle icon next to your cursor before clicking — if you do not see it, you are not hovering precisely enough over the first anchor point.

---

### Step 9: Adjust Your Vector Shape Using the Direct Selection Tool

One of the biggest advantages of working with vectors is that you can always go back and fine-tune your shape after drawing it. The **Direct Selection Tool** lets you click on any individual anchor point and drag it to a new position, reshaping that part of your vector without affecting the rest of the shape or having to redraw anything from scratch.

1. **Select** the *Direct Selection Tool* by pressing `A` on your keyboard — your cursor will change to a white arrow to indicate the tool is active
2. **Click** any anchor point on your shape to select it — the selected anchor point will turn solid blue, while unselected points remain hollow
3. **Drag** the selected anchor point to a new position to adjust that portion of your shape — the surrounding paths will automatically update to follow the moved point

!!! note
    The small hollow dots on your shape are **anchor points**, and the lines connecting them are **paths**. Moving a single anchor point only affects the paths directly connected to it, so you can make very precise adjustments to just one part of your shape without disturbing the rest of the outline.

!!! note
    If you cannot see the anchor points, click directly on the shape first with the Direct Selection Tool to make them visible.

---

### Step 10: Save Your File

To preserve all of your vector layers and shapes, you must save your file in Photoshop's native format. Saving in any other image format will permanently flatten your layers into regular pixels, meaning you will no longer be able to select, resize, or edit your vector shapes in the future.

1. **Click** [File] → [Save As]
2. **Set** the format to `.psd` — this is Photoshop's native file format that preserves all layers, vector data, and edit history
3. **Click** [Save] to save your file

!!! warning
    Saving as `.jpg` or `.png` will permanently flatten your vector layers into pixels and you will lose the ability to scale or edit your shapes without quality loss. Always save your working file as `.psd`, and only export to `.jpg` or `.png` when you have a finished version that you no longer need to edit.
