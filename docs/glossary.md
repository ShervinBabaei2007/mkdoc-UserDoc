# Glossary

This glossary defines key terms used throughout the Non-Destructive Editing and Vector Shapes sections of this guide. Terms are listed in alphabetical order.

| Term | Definition |
|---|---|
| **Adjustment Layer** | A special layer that applies color or tone changes (such as Hue/Saturation) to all layers below it without permanently altering any pixels. Adjustment Layers can be modified, hidden, or deleted at any time. |
| **Anchor Point** | A small dot placed on the canvas when using the Pen Tool or Shape Tool. Anchor points define the corners and curves of a vector path and can be moved at any time using the Direct Selection Tool. |
| **Background Layer** | The default bottom layer created when you open or create a new file in Photoshop. The Background layer is locked by default and should never be edited directly — always duplicate it first to protect the original. |
| **Close Path** | The action of connecting the last anchor point of a Pen Tool path back to the first anchor point, creating a fully enclosed shape. A closed path is required before a vector shape can be filled with a solid color. |
| **Curve Handle** | A directional control that appears when you click and drag with the Pen Tool to place a curved anchor point. Dragging the curve handle changes the direction and intensity of the curve between two anchor points. |
| **Destructive Editing** | Any edit that permanently changes the pixels of an image, such as applying Hue/Saturation directly via [Image] → [Adjustments]. Destructive edits cannot be undone once the file is saved and closed. |
| **Direct Selection Tool** | A Photoshop tool (shortcut: `A`) used to select and move individual anchor points on a vector shape. Unlike the regular Selection Tool which moves the entire layer, this tool lets you adjust just one part of a vector path. |
| **Duplicate Layer** | A copy of an existing layer created by pressing `Ctrl + J` or right-clicking the layer and selecting [Duplicate Layer]. Working on a duplicate protects the original layer from accidental changes. |
| **Fill** | The solid color, gradient, or pattern applied to the inside of a vector shape. The fill color is set in the top options bar before or after drawing a shape using the Shape Tool. |
| **Gaussian Blur** | A filter found under [Filter] → [Blur] → [Gaussian Blur] that softens an image by blending pixel values together. When applied to a Smart Object, it becomes a Smart Filter that can be adjusted or removed at any time. |
| **Hue/Saturation** | A color adjustment that controls the hue (color tone), saturation (color intensity), and lightness of an image. It can be applied destructively via [Image] → [Adjustments] or non-destructively via an Adjustment Layer. |
| **Layers Panel** | The panel in Photoshop (usually on the right side of the screen) that displays all layers in the current document. Each layer can be hidden, reordered, renamed, or deleted from this panel. |
| **Non-Destructive Editing** | An approach to editing that preserves the original image pixels at all times using Adjustment Layers, Smart Objects, and Smart Filters. All non-destructive changes can be modified or removed without permanently affecting the original image. |
| **Path** | A line drawn between two anchor points using the Pen Tool. Paths define the outline of a vector shape and can be straight or curved depending on how the anchor points were placed. |
| **Pen Tool** | A Photoshop tool (shortcut: `P`) used to draw custom vector shapes by manually placing anchor points on the canvas. Clicking creates sharp corner points, while clicking and dragging creates smooth curved points. |
| **PSD** | The native file format for Adobe Photoshop (`.psd`). Saving as PSD preserves all layers, Smart Objects, Adjustment Layers, and vector data so they remain fully editable. |
| **Raster** | An image made up of individual pixels arranged in a grid. Raster images lose quality and become blurry when scaled up because there are a fixed number of pixels — most photographs are raster images. |
| **Shape Layer** | A vector-based layer created when you draw a shape using the Shape Tool or Pen Tool. Shape Layers store the shape as a mathematical path rather than pixels, meaning it can be resized to any size without quality loss. |
| **Shape Tool** | A Photoshop tool (shortcut: `U`) that lets you draw pre-built vector shapes such as rectangles, ellipses, triangles, stars, and polygons. The top options bar must be set to **Shape** mode for the result to be saved as a vector layer. |
| **Smart Filter** | A filter applied to a Smart Object layer that remains fully editable after it is applied. Smart Filters appear underneath the Smart Object in the Layers panel and can be double-clicked to adjust or removed entirely at any time. |
| **Smart Object** | A special layer type that wraps the original content inside a protected container, preventing permanent changes. Filters applied to a Smart Object become Smart Filters, and the original image data is never altered. |
| **Stroke** | An outline or border applied to the outside edge of a vector shape. The stroke color and thickness are set in the top options bar alongside the fill color when using the Shape Tool. |
| **Vector** | A type of graphic defined by mathematical paths and anchor points rather than pixels. Vectors can be scaled to any size without losing sharpness or quality, making them ideal for logos, icons, and illustrations. |
